Description

The Pathfinding Visualizer project is a ReactJS application that utilizes Dijkstra's algorithm to find the shortest path between a specified start and end node in a grid. Users can interact with the grid, setting up blockages to simulate real-world scenarios. This application provides a visual representation of the algorithm's process, enhancing understanding and intuition for pathfinding in graphs.
